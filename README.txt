OPERACJA FENIKS v2.0

WAŻNE:
W tej paczce strona główna nazywa się poprawnie: index.html

Jak wrzucić:
1. Wejdź w repozytorium Operacja-Feniks.
2. Usuń stary plik indeks.html albo zostaw go, ale najważniejsze jest, żeby był plik index.html.
3. Wgraj wszystkie pliki z tej paczki:
   - index.html
   - style.css
   - filip.html
   - antek.html
   - marcel.html
   - monika.html
   - grzegorz.html
4. Wejdź w Ustawienia -> Strony.
5. Ustaw: Wdrażanie z gałęzi / główny / /(źródło).
6. Zapisz.
