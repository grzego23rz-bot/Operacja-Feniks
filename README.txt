1. Załóż konto na github.com.
2. Utwórz nowe repozytorium, np. operacja-feniks.
3. Wrzuć wszystkie pliki z folderu operacja_feniks_site.
4. Wejdź w Settings -> Pages.
5. Wybierz branch main i folder root.
6. Po chwili dostaniesz link.
